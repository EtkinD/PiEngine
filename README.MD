# PiEngine

[Türkçe](#TR) / [English](#EN)

## TR
PiEngine, `pygame` kütüphanesini sarmalayıcı bir kütüphanedir. Python ile oyun geliştirerek küçük yaş gruplarına python öğretme amacıyla oluşturulmuştur.

## EN

PiEngine is a wrapper library for the `pygame` library. It was created to teach Python to young age groups by developing games with Python.
